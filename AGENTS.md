# CreditMiners AI Entry Point

Welcome to the CreditMiners project.

This repository contains additional guidance for AI coding assistants and human contributors.

## Before Making Changes

Please read the following documents in order:

1. `.ai/AGENTS.md` — Primary instructions for AI agents.
2. `.ai/PROJECT_CONTEXT.md` — Project overview and current status.
3. `.ai/PRODUCT_REQUIREMENTS.md` — Functional and non-functional requirements.
4. `.ai/ARCHITECTURE.md` — System architecture and module responsibilities.
5. `.ai/API_CONTRACT.md` — Planned REST API specification.
6. `.ai/UI_UX_GUIDELINES.md` — User interface and experience guidelines.
7. `.ai/CODING_STANDARDS.md` — Coding conventions and best practices.

## Purpose

These documents ensure that all contributors—human or AI—work from the same understanding of the project's goals, architecture, and development standards.

If any implementation changes affect the documented behavior or APIs, update the relevant documentation as part of the same change.
